Pneumonia Detection using Deep Learning

This project uses a deep learning model to detect pneumonia from chest X-ray images.

🔍 Features
- Image classification using CNN / ResNet50
- Preprocessing and augmentation
- Model evaluation (accuracy, precision, recall, F1-score)

📁 Files
- train_pcode.ipynb → Training notebook
- test_pcode.ipynb → Testing & evaluation


🚀 Future Improvements
- Deploy as web app
- Improve accuracy with ensemble models
